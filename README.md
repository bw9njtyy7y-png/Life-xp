# Life XP

Personal PWA for tracking real-life achievements as XP, levels, and reward cards.

## Files

- `index.html` - app shell
- `styles.css` - interface and character visuals
- `app.js` - XP, levels, rewards, and local storage logic
- `manifest.webmanifest` - PWA install metadata
- `sw.js` - offline cache
- `icons/` - app icons

## Deploy

This is a static site. It can be hosted on GitHub Pages, Cloudflare Pages, Netlify, Vercel, or any static web host.

For GitHub Pages:

1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Open repository Settings.
4. Go to Pages.
5. Set Source to `Deploy from a branch`.
6. Select `main` and `/root`.
7. Open the generated Pages URL.

The app stores data in the browser on the device where it is used.
